1. office:apache open office
    * [Download](https://www.openoffice.org/download/):注意選擇下載版本, 我是ubuntu系統, 是debian系的,所以我選擇Linux 64-bit (x86-64)(DEB)下載(chinese simplified)

* 接下來是vscode:

```bash
wget -q https://packages.microsoft.com/keys/microsoft.asc -O- | sudo apt-key add -
sudo add-apt-repository "deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main"
sudo apt update && sudo apt install code
```

由於源在海外,整個過程可能需要十幾分鍾.當然,到官網下載他們指定的的安裝包也可以吧(我沒試過):[安裝包](https://go.microsoft.com/fwlink/?LinkID=760868)

* 中文輸入的問題, ubuntu是自帶漢語輸入的, 設置辦法:

1. setting--region&language--manage更新語言包, 在language for裏面滾動鼠標確認有漢語/中文
2. Keyboard--input_sources--"+"--Chinese(china)--chinese(intelligent pinyin)
3. reboot

我是昨天設置的漢語輸入,這些過程沒有實操檢驗, 但是憑藉網上的教程我也比較快的解決了, 現在回憶起來大概是這個流程

注意,101中提到應使用`apt install ./*.deb`代替`dpkg -i *.deb`安裝
