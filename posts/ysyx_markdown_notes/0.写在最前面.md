# 一生一芯从这里开始

在很久之前，我就有自己学习的想法了，即使父母和老师抑制不住的热情时常不满于我这个想法，也很难获得同学们的共识，但是学习时的平静的愉悦，有如清冽甘泉入口，令人久久不能忘怀，并且使人愿意为之克服取得它之路上的所有困难。青春之泉的故事流传了几个世纪，也从未有人能发现那样神奇的液体。在哪里能找到呢？其实古人已经找到了答案吧，“为有源头活水来”，永葆青春的秘密就是源源不断的新知。

可是，时间仍然是紧迫的,"予少時讀書，一見輒能誦。暗疏之，亦不甚失。然負此自放，喜從滑稽飲酒者遊。旬朔之間，把卷無幾日。故雖有強記之力，而常廢於不勤。","噫！少而不勤，無如之何矣。長而善忘，庶幾以此補之。"精骑集里这几句话我一直记得很深刻,比起单说空洞的珍惜时间,我感到了作者对自己才能被自己浪费的懊悔,和对知识的渴望。现在还来得及吗？秉烛夜游也好。

这里是预学习的第一篇，也是整个计划的第一篇，我仍然会采用计划+随笔的办法来对待这项任务，这是我迄今为止采用过的最高规格，最复杂，最认真的办事方法了，是整个高中方法论探索的结果，也是我交给中国应试教育的个人应对最终方案。下一个方法论的诞生，可能要到工作很多年之后了吧。
