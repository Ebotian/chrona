### Project Codename: Chrono-Stasis — Synthwave Vector Constellation Design Bible

**版本：** 2.0
**设计师：** GitHub Copilot · Senior Visual & Interaction Designer
**日期：** 2025 年 10 月 31 日

> 全站一切线条、光束、粒子均遵循“零面积移动 (Zero-Area Motion)”原则：线条只可沿自身路径伸缩或弯折，严禁以填充、淡出或扫面方式出现与消失；所有节点必须通过一条或多条连续不自交的线段回连屏幕边缘，确保整体一体化。

---

### 第一章 · 核心哲学与交互准则

1. **宇宙黑场开篇**：以纯黑画布开启旅程，线条先于色彩显现，构筑叙事骨架。黑场过渡为深空渐层的过程完全由线条驱动。
2. **零面积移动**：任何组件的出现、转场、解构都依赖线段的展开与回收；动画的所有面积均保持零值，仅允许线宽/亮度变化。
3. **合体线网**：全站建立一个“主干线网”，文字、按钮、粒子、面板均以支线挂载，避免孤岛几何，确保能量流动感。
4. **Synthwave Retrowave 美学**：高对比度星云背景 + 霓虹线框城市 + 透视网格，注入 70-90 年代太空朋克地标轮廓。
5. **多感官叙事**：视觉（星云/电波/网格）、听觉（Lost Years《Amplifier》）、微动效（指针扰动、波形震荡）共同组织节奏。
6. **内容为核**：再炫目的皮肤都服务于 Markdown 技术档案的阅读效率；索引、正文、关联推荐需要在同一线网中自然衔接。
7. **性能先导**：线条复用、网格共享顶点、粒子数量可回落，确保老设备可在降级模式下保持叙事完整。
8. **无障碍守恒**：动效可一键降至静态线框，声效可静音但仍保留播放状态提示，文字对比度满足 AA+。

### 第二章 · 视觉识别系统

#### 2.1 Chrono Glyph（常驻标识）

- **结构**：一条折线自右下角入场后连出三环节点，最终收束为左上角的 Chrono Glyph。环心点亮代表站点心脏。
- **动态**：环线每 120 秒沿自身路径回缩再重绘，形成轻扰动。悬停时三环瞬时闭合并沿原路径外翻，合乎零面积概念。
- **构网**：Glyph 的所有顶点与主线网同源，允许后续导航与状态提示借同一条线传递。

#### 2.2 色彩体系

| 层级     | 色名                | Hex                    | 应用                                 |
| -------- | ------------------- | ---------------------- | ------------------------------------ |
| 宇宙基底 | Event Horizon Black | #010005 → #040c20 渐变 | 黑场至深空的过渡，渐变沿线网方向铺展 |
| 星云高光 | Nebula Violet       | #6f3bff                | 背景星云热点、信息凸显               |
| 线框主色 | Plasma Cyan         | #00f7ff                | 主干线网、按钮轮廓、波形光束         |
| 线框次色 | Mag Ridge Pink      | #ff2d92                | 电波天际线刷新、高亮提示             |
| 支撑色   | Carbon Gray         | #3a4252                | Markdown 文本、次要线段              |
| 告警色   | Solar Orange        | #ff6a00                | 错误提示、重要交互                   |

- **噪点处理**：背景星云叠加 <2% 粒状噪点；天际线刷新时瞬时提升至 3%，随即回落。
- **辉光控制**：辉光仅沿线方向扩大 6-18px，不允许面状扩散。

#### 2.3 字体排印

- **标题**：`Monument Extended` 或 `Neue Machina`，纯大写，字间距 -0.04em。
- **正文**：`Inter` 系列，正文 18px/1.7，宽度 <= 650px。
- **代码**：`Fira Code` 17px，启用连字。
- **动效**：标题字形沿主线顺序点亮，缩放范围 0.85→1.0；正文段落由线框框出后才显现文字，文字与框的动效同步。

#### 2.4 线条与图形语汇

- **线宽**：默认 2px，Glyph 与网格 3px，支线 1px。
- **角度**：优先使用 0° / 30° / 60° / 120° 的几何折角，保持 retrowave 透视感。
- **节点**：圆、菱形、三角三种，依据信息层级选择。节点扩展时沿线延展成胶囊，不形成独立面。

### 第三章 · 场景布局与组件架构

#### 3.1 主场景流程

1. **黑场序章**：三条线从屏幕角落缓慢入场，构成地平弧与前景网格骨架。
2. **宇宙幕布**：线条稳定后，星云背景沿主线方向渐显，星团节点与时间轴将在此挂载。
3. **电波天际线**：三层折线共享基线，依次“回缩→重绘”形成 synthwave 城市轮廓，嵌入经典地标外形。
4. **入口叙事**：Slogan、CTA、音频条及 Ambient 节点沿主线依序生成，满足零面积动效。
5. **索引星座**：主页下半部分构成星座式索引，所有年份/主题沿径向主线排布。
6. **文章长卷**：进入正文后，线段延展成 Scene Panels、Pulse Terminal、Dialogue Capsules 等组件。

#### 3.2 主页组件

- **Chrono Initiation Grid（升级 Preloader）**：主线点亮三环 Glyph，电流沿线唤醒网格与天际线。
- **Background Universe**：R3F 渲染 Webb 风格星云；所有粒子沿可追踪线段漂移。
- **Synthwave Skyline**：折线模仿西雅图太空针塔、世贸双塔、约翰·汉考克中心等轮廓；刷新时保持零面积回缩。
- **Slogan Cluster**：`Chrono-Stasis Vector Log` 标题沿主线分段排布，指针位置影响背后三角光束。CTA 为与主线共边的长条按钮，点击后沿线收缩并引导滚动。
- **Floating Audio Rail**：位于左下网格节点的长条播放组件，含曲名、进度光点、模式切换按键。进度条以光点沿线滑动表示。
- **Ambient Toggle**：右下角放置 Amplifier Node，波形沿支线上下微震。

#### 3.3 索引与导航

- **Constellation Index**：6 条径向主线，节点表示年份/专题；展开时条目沿垂线落下，条形卡片与主线共边。
- **Timeline Beacon**：左侧垂直主干线贯穿全屏，节点显示年份，可即时跳转，同时维持线网连通。
- **HUD Menu**：右上角节点展开后沿主线折叠成 80% 高导航板。

#### 3.4 内容呈现

- **Cinematic Header**：顶部线带展示文章元数据；滚动至阈值主线回缩，标题沿线滑至侧栏。
- **Scene Panels**：正文段落由主线延伸成开口矩形框，框收束时正文淡出。
- **Pulse Terminal**：代码块放置于倾斜矩形线框，复制按钮与主线相连。
- **Dialogue Capsules**：引用块沿主线伸出半透明胶囊，回收时沿线缩回。
- **Interactive Keyframes**：3D/数据模块挂在主线节点，滚动驱动沿线展开；提供“Skip”支线归位。
- **Image Lab**：Lightbox 打开时主线向两侧拉开，形成中央展示台。
- **Echo Chamber / Author Glyph**：评论区与作者卡片均使用沿主线延展的长条，以波形/扫描线表现交流感。

### 第四章 · 动效语言（零面积运动版）

| 场景        | 触发           | 描述                                 | 时长   | 缓动           |
| ----------- | -------------- | ------------------------------------ | ------ | -------------- |
| 黑场开启    | 0s             | 三条主线同时沿边缘伸展并弯折         | 1.6s   | easeInOutCubic |
| Glyph 点亮  | 1.6s           | 三环顺序绘制，收束为左上角常驻标识   | 1.2s   | easeOutQuad    |
| 天际线刷新  | 每 12s         | 折线沿基线回缩后重绘，配 2 帧 glitch | 0.9s   | easeInOutExpo  |
| Slogan 激活 | Preloader 完成 | 标题分段沿线伸展，CTA 充能光带拉开   | 0.8s   | easeOutBack    |
| 索引展开    | 节点 hover     | 节点沿线放大，条目垂直支线延伸       | 0.4s   | linear         |
| 文章滚动    | 阅读中         | Scene Panel 线框延长→收回，文字渐显  | 视滚动 | custom         |

**音画同步**：Lost Years《Amplifier》四小节触发天际线刷新，八小节更新星云高光，十六小节 CTA 辉光增强。

### 第五章 · 用户旅程

1. **首次拜访**：黑场 → 主线点亮 → 背景星云 → Slogan → 进入索引星座。
2. **查阅历史**：通过 Timeline Beacon 或索引节点选择年份，沿主线滑向对应 Markdown 文章。
3. **深度阅读**：文章中穿梭于 Scene Panel、Pulse Terminal、Image Lab；关键帧模块提供滚动或 Skip 控制。
4. **持续探索**：文章尾部 Echo Chamber 与 Author Glyph 引导评论、分享与下一篇推荐。
5. **回访**：离开后再次进入可直接触发 HUD Menu 或某个索引节点，所有线条保持熟悉布局。

---

### 第六章 · 响应式策略

- **Desktop ≥1280px**：完整星云、三层天际线、全网格与粒子。
- **Tablet 768-1279px**：粒子减 40%，索引星座改为两列径向排列，音频条缩短但保持线网连接。
- **Mobile ≤767px**：线网简化为单层透视，天际线刷新频率减半，动效高度减低；音频条与 CTA 合并成一条底部线段。
- **prefers-reduced-motion**：保留线条结构，但停止刷新、粒子静止、音频默认静音。

---

### 第七章 · 无障碍与包容性

1. 所有文本与背景对比度 ≥ 4.5:1；线条辉光不掩盖文字。
2. 提供“静音”“简化动效”开关，开关本身沿主线挂载，状态清晰可见。
3. Tab 序遍历按主线顺序进行，焦点状态以电浆蓝外描线实现。
4. 图片、星图节点提供文字描述；音频组件播报当前曲目与状态。
5. 字体覆盖中英双语，排版自动调整字重与字距。

---

### 第八章 · 视觉资产与素材库

- **星云纹理**：整理 Webb 深场素材，制作三层噪点贴图。
- **城市线稿库**：复刻 70-90 年代代表性摩天楼轮廓（太空针塔、双子塔、John Hancock Center、Mode Gakuen Cocoon、NASA VAB、金茂大厦等）。
- **音频素材**：Lost Years《Amplifier》作为常驻，另备 2 首备用曲目；音效库保持低频 synthwave 调性。
- **线条预设**：提供主线、支线、弧线模板，方便衍生新模块时复用。
- **Moodboard**：以《星球大战》开幕布景、《银翼杀手 2049》城市夜景、Synthwave 艺术家 Beeple 等视觉为参照。

### 第九章 · 模块编排示例

1. **星云主页**：Chrono Initiation Grid → Synthwave Skyline → Slogan → Audio Rail → Constellation Index。
2. **年度专题**：从特定年份节点展开星座，页面内通过 Scene Panels 与 Pulse Terminal 展开年内文章合集。
3. **文章长卷**：Cinematic Header → Scene Panels → Pulse Terminal → Dialogue Capsules → Image Lab → Echo Chamber → Author Glyph。
4. **档案库**：主线延展成网格光廊，节点展示视频、图像或代码档案，Lightbox 以线条开启与关闭。

### 第十章 · 协作与迭代建议

- **交付物**：Figma 线网模板 + Lottie 零面积动效 JSON + GLTF 天际线模型 + Tone.js 音频参数。
- **验收标准**：动效帧率 ≥ 50fps；线条数量在预算内；降级模式保持完整叙事；无障碍检查通过。
- **后续拓展**：
  1. 实时数据流：从代码仓库或 RSS 抓取动态，沿主线粒子带滚动。
  2. AI 叙事助手：在索引星座旁加入对话节点，为读者推荐冷门条目。
  3. 自定义皮肤：允许用户挑选不同年代的城市线稿与配色，线条体系继续沿主网结构生成。

**结语**

Chrono-Stasis 2.0 以零面积移动和 synthwave retrowave 语言为基石，将技术博客塑造成一座脉动的光谱档案馆。每一根线条都是叙事导线，每一篇 Markdown 文档都被嵌入星际轨道，读者在其中穿行，既能快速检索，也能沉浸式体验时间与代码的纵深。\*\*\* End Patch
